Info

Help

SolarOne Web calculates and visualizes solar data for places using browser-only storage.

Quick Start

  • Use World Map to view places and select/deselect them.
  • Use Add Place to geocode a location, adjust coordinates, and save.
  • Use Graphs to render Rise/Set, Max Elevation, and Length of Day.
  • Use Solar Sector to inspect sunrise-to-sunset azimuth sector by day.

Data and Storage

  • All data is stored in your browser (localStorage), no backend required.
  • Starter places are seeded on first run; use Restore Starter Places if needed.
  • Yearly solar values are precomputed and cached per place/year for performance.

No-key Mode Notes

  • Maps use OpenStreetMap tiles (no billing key).
  • Geocoding/timezone lookup uses public services and may be rate limited.
  • Satellite map style is intentionally not included in no-key mode.

Accessibility

  • Status and feedback messages are announced via live regions.
  • Forms use labels and native controls for keyboard and screen-reader support.
  • Use browser zoom and responsive layout for readability on different devices.
